# 🚀 CodeForge

**CodeForge** is a modern online coding platform that offers an AI-enhanced custom compiler, a robust backend hosted on AWS EC2, and a feature-rich practice environment for programmers. Whether you're a beginner or preparing for interviews, CodeForge gives you the tools to improve your coding skills, track progress, and get intelligent code suggestions.

---

## ✨ Features

### 🧠 AI-Supported Custom Compiler
- Built-in support for multiple programming languages.
- AI suggestions and error hints for faster debugging and learning.
- Secure sandboxed execution for user-submitted code.

### 🔙 Backend Powered by AWS
- Dockerized microservices hosted on an EC2 instance.
- Scalable architecture for handling simultaneous submissions.
- REST API integration for frontend-backend communication.

### ❓ Practice Questions with Test Cases
- Curated problem set with increasing difficulty levels.
- Hidden and visible test cases to mimic real-world platforms.
- Instant feedback on submission.

### 👤 User Profiles & Progress Tracking
- Register/Login system with session management.
- Track problems attempted, solved, and accuracy rate.
- Personalized dashboard with coding stats.
